Description

Cette formation intensive en développement web full stack permet d'acquérir toutes les compétences nécessaires pour concevoir, développer et déployer des applications web complètes. Les participants maîtriseront à la fois les technologies front-end (HTML, CSS, JavaScript, React) et back-end (Node.js, Express, MongoDB).

Notre approche pédagogique repose sur une alternance entre théorie et pratique, avec de nombreux exercices et projets concrets. À l'issue de la formation, les participants seront capables de développer des applications web complètes de A à Z.

Programme détaillé

Module 1: Fondamentaux du Web (30 heures)

  • HTML5 et sémantique web
  • CSS3 et animations modernes
  • JavaScript ES6+

Module 2: Front-end Avancé (45 heures)

  • React.js fondamentaux
  • Gestion d'état avec Redux

Prérequis

  • Bonne maîtrise de l'outil informatique
  • Connaissances de base en algorithmique (un plus)
  • Logique et esprit analytique
