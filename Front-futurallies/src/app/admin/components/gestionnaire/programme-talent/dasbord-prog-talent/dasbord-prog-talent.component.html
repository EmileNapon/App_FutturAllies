Résumé:

Formation intensive en développement web couvrant les technologies front-end et back-end pour devenir développeur full-stack.

Description complète:

Cette formation vous permettra de maîtriser les technologies clés du développement web moderne. Vous apprendrez HTML, CSS, JavaScript, React pour le front-end, ainsi que Node.js, Express et MongoDB pour le back-end. La formation inclut des projets pratiques et une certification finale.
